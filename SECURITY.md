# Security

## Supported versions

This repository ships example Terraform for Azure Red Hat OpenShift (ARO). Only the **latest commit on the default branch** is supported for security updates to these files. Tagged releases are historical snapshots; upgrade to the latest tag or `main` when addressing issues.

Infrastructure you deploy with this code is **your** Azure subscription and responsibility: patch clusters and dependencies per Red Hat and Microsoft guidance.

## Reporting a vulnerability

**Do not** open a public GitHub issue for undisclosed security vulnerabilities in this repository.

Please use **[GitHub Security Advisories](https://github.com/rh-mobb/terraform-aro/security/advisories/new)** to report security issues in this Terraform project (private disclosure to maintainers).

For **product** vulnerabilities in ARO, OpenShift, or Azure, report through the appropriate vendor channels (e.g. [Red Hat Product Security](https://access.redhat.com/security/team/contact/), [Microsoft Security Response Center](https://www.microsoft.com/en-us/msrc)).

## Response

Maintainers will acknowledge receipt as schedules allow. This is a community example repository; there is no formal SLA. Critical issues will be prioritized when possible.
